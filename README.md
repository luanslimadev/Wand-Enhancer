## 📖 Sobre

O **Wand Enhancer** é uma ferramenta poderosa e intuitiva. Muitos usuários têm dificuldade em baixar o código-fonte, instalar as dependências e compilar o projeto por conta própria. 

Pensando nisso, disponibilizo aqui a versão já compilada (formato `.exe`), facilitando o acesso para quem deseja apenas baixar e usar o programa imediatamente, sem complicações técnicas.

## 🚀 Como usar (Instalação)

1. Vá até a seção **Releases** (ou clique no link de download fornecido).
2. Baixe o arquivo `WandEnhancer.exe` (ou o arquivo `.zip` disponibilizado).
3. *(Opcional)* Extraia o arquivo caso esteja zipado.
4. Execute o aplicativo dando um duplo clique.
   * *Nota: Como o aplicativo é compilado de forma independente (não possui um certificado digital pago), o Windows Defender (SmartScreen) pode exibir um aviso de "Aplicativo não reconhecido". Basta clicar em "Mais informações" e depois em "Executar assim mesmo".*
